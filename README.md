# 医院物资管理系统 java624

## 项目概述

医院物资管理系统是基于`SpringBoot`、`MyBatis`和`Vue`的整合开发，旨在为医疗机构提供一个高效、稳定的物资管理解决方案。

## 技术栈

### 后端技术

- SpringBoot
- MyBatis

### 前端技术

- Vue

## 运行环境

- 开发工具：IDEA / Eclipse
- 数据库：MySQL 5.7
- 运行方式：SpringBoot

## 功能模块

系统主要包括但不限于以下功能模块：

- 账户管理
- 物资入库
- 物资出库
- 库存管理
- 报告生成

## 系统角色

- 超级管理员
- 员工

## 部署步骤

1. 环境准备：确保MySQL 5.7已安装且可用。
2. 数据库构建：运行数据库脚本以建立初始数据库和表结构。
3. 项目构建：使用Maven或相应的构建工具编译项目。
4. 配置修改：根据实际环境，调整配置文件。
5. 启动服务：运行SpringBoot应用程序。
